# Epiderm

Epiderm is an AI-powered skincare analysis platform designed to help users better understand their skin through image analysis, personalized recommendations, and long-term progress tracking.

## Features

- AI-powered skin analysis
- Body-part specific assessments
- Personalized skincare routines
- Skin Journey tracking
- Before & After comparisons
- Dark Mode
- Privacy-first image handling
- Product recommendations with explanations
- Confidence-based analysis

## Mission

Epiderm was created to make skincare guidance more accessible, understandable, and personalized through artificial intelligence.
